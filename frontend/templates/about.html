Project overview

A modern flood intelligence platform.

Rising Waters is an AI-powered product experience focused on proactive flood risk estimation, preparedness workflows, and high-quality decision support.

The experience combines thoughtful UI, responsive interactions, and a structured prediction journey designed for enterprise-style demos and stakeholder presentations.

Technology stack

  • Flask for application routing
  • Bootstrap 5 for modern UI patterns
  • JavaScript for validation and animation
  • CSS variables and responsive design

Machine learning workflow

The workflow begins with environmental features, runs through preprocessing and model scoring, and concludes with a polished probability-based recommendation.

Dataset information

The experience is designed around rainfall, temperature, humidity, and seasonal climate signals that are commonly used in flood forecasting studies.

Algorithms used

  • Decision Tree
  • Random Forest
  • KNN
  • XGBoost

Developer information

This frontend experience was crafted to feel premium, accessible, and recruiter-ready while staying tightly integrated with the Flask backend.
