# UI and Visual Documentation

This folder is reserved for visual documentation and screenshots of the Rising Waters web application.

## Purpose
- Store responsive UI screenshots for mobile, tablet, and desktop layouts.
- Provide visual evidence of the prediction workflow, input validation, and result pages.
- Support delivery artifacts for audit and review.

## Current Coverage
- The web UI is built using a responsive Bootstrap-based layout.
- Input forms, prediction results, and navigation screens are designed for mobile and desktop.

## How to Add Screenshots
1. Capture browser screenshots for key pages:
   - Home page
   - Predict input page
   - Result page
2. Save the images in this folder with descriptive names, such as `predict-form.png` or `result-page-mobile.png`.
3. Optionally, add a `screenshots.md` file with notes on the device and browser used.
